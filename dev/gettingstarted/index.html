Installation

  1. Install Julia, the latest versions of ACE.jl require v1.3 but should also work ok with v1.4, future versions will likely require v1.4.
  2. Install the MolSim registry; from the Julia REPL, switch to package manager ] and then run
registry add https://github.com/JuliaMolSim/MolSim.git
  1. Install some important registered packages; from Julia REPL / package manager:
add PyCall IJulia     # add more important packages from General registry
add JuLIP ACE ASE   # maybe add other packages from MolSim registry
  1. For fitting, need to install also IPFitting.jl,
add IPFitting

(Keep fingers crossed and hope it will be compatible with the current version of ACE.jl...)
